# Development

## Prerequisites

- Node.js 22.x

## Getting Started

1. Clone this repository
2. Install dependencies:

```bash
npm install
```

3. Run in dev mode

```bash
npm run dev
```
4. Browse to http://localhost:3000/

## Deployment

Generate the production build using

```bash
npm run build
```
